🎓 WHY THIS ERROR HAPPENED
═══════════════════════════════════════════════════════════════════════

pip tried to build ONNX from source (using CMake) because it couldn't
find a pre-built wheel for your Python version. Building requires
Visual Studio C++ compiler which you don't have configured.

THE FIX: Use pre-built wheels or Conda (which handles binaries better)


🐍 PYTHON VERSION
═══════════════════════════════════════════════════════════════════════

Check yours:  python --version

Best:         Python 3.10 or 3.11
Acceptable:   Python 3.8, 3.9
Problematic:  Python 3.13+ (too new, limited wheels)
              Python 3.7- (too old)

If you have 3.13+, use Conda or downgrade to 3.11
