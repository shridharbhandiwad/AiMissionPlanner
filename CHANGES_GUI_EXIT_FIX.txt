PROBLEM FIXED
================================================================================

The trajectory GUI application was exiting silently after printing "Checking
dependencies..." without showing any error message or launching the GUI window.

Users had no way to diagnose what was wrong.

================================================================================

================================================================================
COMMON SOLUTIONS
================================================================================

Issue: PyOpenGL not installed
Fix:   pip install PyOpenGL PyOpenGL_accelerate

Issue: Graphics driver problems
Fix:   Update drivers OR set LIBGL_ALWAYS_SOFTWARE=1

Issue: Corrupted packages
Fix:   pip uninstall PyQt5 pyqtgraph PyOpenGL
       pip install PyQt5 pyqtgraph PyOpenGL PyOpenGL_accelerate

Issue: Python 3.12+ compatibility
Fix:   Use Python 3.11: conda create -n env python=3.11
