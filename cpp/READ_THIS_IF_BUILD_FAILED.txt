STEP 2: Install missing tools (if any)

If the check shows missing tools, install them:

    Check "Add to PATH" during installation

    2. Run: pacman -S mingw-w64-x86_64-gcc mingw-w64-x86_64-make
    3. Add to PATH: C:\msys64\mingw64\bin
    4. Restart your terminal

┌────────────────────────────────────────────────────────────────────┐
│                    COMMON ISSUES                                   │
└────────────────────────────────────────────────────────────────────┘

Issue: "CMake not found"
→ Install CMake and make sure "Add to PATH" is checked
→ Restart your terminal after installation

Issue: "mingw32-make not found" or "g++ not found"
→ Install MSYS2 and MinGW: https://www.msys2.org/
→ Run: pacman -S mingw-w64-x86_64-gcc mingw-w64-x86_64-make
→ Add C:\msys64\mingw64\bin to PATH
→ Restart your terminal

Issue: "Build folder won't delete"
→ Close any programs that might have files open
→ Run as Administrator
→ Or manually delete: rmdir /s /q build

Issue: "Still getting errors"
→ Read WINDOWS_BUILD_FIXED.md for detailed troubleshooting
→ Check BUILD_FIX_SUMMARY.md for technical details
